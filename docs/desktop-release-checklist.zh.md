# 桌面安装版发布验收清单

适用版本：`0.2.x` 第一阶段自用未签名版本。

## 自动化门槛

- [ ] `npm ci` 成功。
- [ ] 全部单元、接口和本地模拟发布测试通过。
- [ ] TypeScript 三套配置检查通过。
- [ ] Windows NSIS 安装包构建成功并生成 SHA-256。
- [ ] macOS Universal DMG 构建成功并生成 SHA-256。
- [ ] 本地模拟发布的错别字分支只点击“全面检测”。
- [ ] 本地模拟发布的直接检测分支只点击“全面检测”。
- [ ] 模拟发布测试没有请求 `fanqienovel.com`。

## Windows 10/11 x64

- [ ] 在没有启动 Node.js、Vite 或 Express 服务的电脑上安装。
- [ ] SmartScreen 出现时，“更多信息 → 仍要运行”的说明准确。
- [ ] 安装目录可选择，桌面快捷方式可正常创建。
- [ ] 应用打开后版本号、应用数据地址、Chrome 资料地址和日志地址正确。
- [ ] 系统文件夹选择器可以选择含中文名称的小说文件夹。
- [ ] 能检测系统 Chrome；没有 Chrome 时给出明确提示且不开始发布。
- [ ] 覆盖安装同版本或新版本后，设置和 Chrome 登录状态仍保留。
- [ ] 应用内卸载预览显示所有完整路径。
- [ ] 取消勾选时保留小说 `.fanqie-publish.json`，应用数据仍删除。
- [ ] 默认勾选时删除登记的 `.fanqie-publish.json`，不删除小说目录和 Markdown 正文。
- [ ] 控制面板/系统设置中的 NSIS 卸载也执行同样的数据询问与清理。

## macOS Intel 与 Apple Silicon

- [ ] DMG 校验通过并能正常挂载。
- [ ] DMG 中同时包含应用和 `Applications` 快捷方式。
- [ ] 主程序与 Electron Framework 都包含 `x86_64` 和 `arm64`。
- [ ] 在没有启动 Node.js、Vite 或 Express 服务时，复制到 Applications 后可独立启动。
- [ ] “隐私与安全性 → 仍要打开”的未签名说明准确。
- [ ] 应用打开后版本号、应用数据地址、Chrome 资料地址和日志地址正确。
- [ ] 系统文件夹选择器可以选择含中文名称的小说文件夹。
- [ ] 能检测系统 Chrome；没有 Chrome 时给出明确提示且不开始发布。
- [ ] 覆盖安装后设置和 Chrome 登录状态仍保留。
- [ ] 应用内卸载可以把 `.app` 移入废纸篓并按选择清理数据。
- [ ] 删除发布记录时不删除小说目录和 Markdown 正文。

## 真实番茄单章验收

- [ ] 已获得用户对本次真实发布的明确授权。
- [ ] 使用专门准备的一章超过 1000 字正文。
- [ ] 只选择一本书、一个章节和一个未来发布时间。
- [ ] 核对章节号、标题、正文、AI 选项“否”、日期和时间。
- [ ] 错别字提示分支点击“提交”，内容检测选择“全面检测”。
- [ ] 点击“确认发布”后，在番茄章节管理页确认状态和时间。
- [ ] 本地 `.fanqie-publish.json` 记录为 `scheduled` 且时间一致。
- [ ] 关闭应用并重新打开，不会重复提交已经定时的章节。

## 发布表述

未完成对应验收时，必须准确写明：

> 已完成自动化测试和安装包构建，尚未在该系统完成真实安装、覆盖更新与卸载验收。

没有取得授权并执行真实番茄单章测试时，必须准确写明：

> 尚未授权或执行真实番茄单章发布，不能表述为真实发布已经验证。

第一阶段安装包未签名时，必须准确写明：

> 当前是未签名自用测试版，不适合作为正式商业版本对外销售。
